Номер: 19063
Заголовок: Не полностью заполняется штамп при генерации файл отчетов в pdf
Важность: Серьезная
Версия: 8.0
Окружение: Windows 10x64(21H1)

Описание:
При генерации файл отчетов Материальная ведомость, Таблица соединений и Перечень элементов в pdf не полностью заполняется штамп. Поля Т.контр., Н.контр. и все даты остаются пустыми.
Воспроизведение:
1. Создать проект
2. Документы - Создать ДЭ
3. Документы - Материальная ведомость - Атрибуты - заполнить все поля
4. Забрать файл отчета на редактирование из базы 
5. Сгенерировать файл отчета
6. Открыть для редактирования
При создании форматок в AutoCAD все поля полностью заполняются.

Скриншот
