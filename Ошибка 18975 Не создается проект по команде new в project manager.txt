Номер: 18975
Заголовок: Не создается проект по команде "new" в project manager.
Важность: Существенная
Версия: 21.0
Окружение: Windows 10x64(21H1)

Описание:
1 Запустить команду SPTABS
2 В командной строке выбрать Project Manager
3 Во вкладке Project Manager нажать new.
Фактический результат: Во вкладке создается пустое окно проекта, все поля заблокированы.
Ожидаемый результат: Во вкладке открывается дерево проекта.

Приложения:
Скриншот
