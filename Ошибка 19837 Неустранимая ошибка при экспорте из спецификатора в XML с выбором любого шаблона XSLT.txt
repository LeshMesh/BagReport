Номер: 19837
Заголовок: Неустранимая ошибка при экспорте из спецификатора в XML с выбором любого шаблона XSLT
Важность: Существенная
Версия: 22.0
Окружение: Windows 11(22H2)

Описание:
1. Открыть стандартный проект nBIM. Модель КМ.
2. Открыть Спецификатор - Мастер экспорта данных
3. Изменить выбранный профиль перед изменением
4. На этапе выбора приложения выбрать Файл в формате XML
5. В поле "Шаблон XSLT для преобразования документа XML" выбрать любой из выпадающего списка.
6. Выполнить экспорт.
Фактический результат: Неустранимая ошибка. Программа закрывается.
По умолчанию поле с шаблоном пустое. Если экспортировать без выбора шаблона, то падения нет, экспорт проходит.

Приложения:
Скриншот, логи
